# MVP-2 Acceptance

## Функциональные критерии

- Уведомления приходят при ключевых событиях заявки.
- Поиск по каталогу корректно работает с новой индексной моделью.
- Менеджер видит расширенные KPI без деградации UX.

## Технические критерии

- Сообщения корректно маршрутизируются через RabbitMQ.
- Повторная обработка событий не приводит к дубликатам.
- Есть наблюдаемость по очередям и consumer errors.
