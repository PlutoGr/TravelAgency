# MVP-2 Scope

## Добавляемый функционал

- Уведомления:
  - email события по заявкам;
  - in-app уведомления;
  - базовые push/web notifications.
- Расширенные фильтры каталога через Elasticsearch.
- Улучшенный менеджерский дашборд (больше KPI и фильтров).

## Не делаем пока

- Полноценный data warehouse.
- Kubernetes и auto-scaling как обязательную часть.
