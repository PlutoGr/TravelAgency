# MVP Roadmap Structure

Эта структура разделяет проект по этапам внедрения:

- `MVP-1-Core` — запуск продукта и основные пользовательские сценарии.
- `MVP-2-Expansion` — расширение функционала и усиление интеграций.
- `MVP-3-Scale` — масштабирование, аналитика и production-hardening.

В каждом этапе есть:

1. `Architecture.md` — архитектура этапа (что включено именно на этом шаге).
2. `Implementation-Spec/` — детальная реализационная спецификация этапа.

Назначение: чётко понимать, что делаем сейчас и что переносим на следующий этап.
