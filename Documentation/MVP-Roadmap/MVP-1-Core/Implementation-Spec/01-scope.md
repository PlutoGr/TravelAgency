# MVP-1 Scope

## Бизнес-цель

Запуск первой версии, где клиент может пройти путь:
поиск тура -> заявка -> коммуникация с менеджером -> подтверждение.

## Функционал в этапе

- Публичный каталог с фильтрами и карточкой тура.
- Регистрация/логин, профиль клиента.
- Создание и просмотр заявок клиента.
- Рабочее место менеджера для обработки заявок.
- Избранное.
- Чат (если успеваем) или комментарии в заявке.

## Отложено на следующий этап

- Отдельный Notification Service.
- Elasticsearch как обязательная часть поиска.
- Продвинутая аналитика и BI.
