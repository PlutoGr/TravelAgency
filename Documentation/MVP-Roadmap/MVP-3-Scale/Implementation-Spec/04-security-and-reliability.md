# MVP-3 Security and Reliability

## Security upgrades

- Политики секретов и ротации ключей.
- Усиленный audit и контроль доступа к аналитическим данным.
- Разделение доступов production/stage/dev.

## Reliability upgrades

- SLO/SLI по ключевым сервисам.
- Chaos/Failure drills для критичных сценариев.
- План аварийного восстановления (RPO/RTO) для сервисов и данных.
