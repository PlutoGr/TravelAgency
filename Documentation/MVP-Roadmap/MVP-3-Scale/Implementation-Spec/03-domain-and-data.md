# MVP-3 Domain and Data

## Аналитическая модель

- `FactUserEvent` (события пользователя)
- `FactBookingEvent` (жизненный цикл заявки)
- `DimTour`, `DimUser`, `DimManager`

## Политики данных

- Retention policy по типам событий.
- Псевдонимизация чувствительных атрибутов в аналитике.
- Версионирование схемы событий.

## Качество данных

- Контроль пропусков ingestion.
- Дедупликация событий по ключу события.
