# MVP-3 Scope

## Целевые направления

- Product analytics и конверсионные воронки.
- Масштабирование сервисов и ресурсов.
- Устойчивость к частичным отказам.

## Функции этапа

- Отдельный analytics-service.
- Отчётность для менеджмента (дашборды, выгрузки).
- Подготовка к multi-instance и auto-scaling сценариям.
